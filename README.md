### Implementando a solução

Para executar o *script* num servidor Linux:
* Copie o arquivo script.sh para máquina (pode ser instância Linux na AWS)
* Dê permissão de execução ao *script*:
  
  `sudo chmod +x script.sh`
* Execute o *script*:
  
  `./script.sh`
